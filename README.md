
# Monika's Rasoi – Cloud Kitchen (Lucknow)

A fast, mobile-first landing page with WhatsApp ordering.

## Scripts
- `npm run dev` – local dev
- `npm run build` – production build (outputs `dist/`)
- `npm run preview` – preview the build

## Deploy (Vercel)
- Import repo on Vercel (Framework: Vite).  
- Build: `npm run build`  
- Output: `dist`

## Edit content
- `src/App.jsx`: brand, menu, delivery areas, hours, address, WhatsApp etc.
- Images: place in `public/` (e.g., `/logo.png`, `/menu/*.jpg`).
